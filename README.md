# Getner & Co. Website

Professional website for Greg Getner's ActiveCampaign consulting practice.

## Project Structure

```
/
├── index.html              # Main website file
├── assets/
│   ├── images/
│   │   └── greg-getner-photo.png
│   ├── css/
│   │   └── styles.css      # Extracted CSS (optional)
│   └── js/
│       └── main.js         # Future JavaScript (optional)
├── README.md
└── .gitignore
```

## Local Development

1. Clone this repository
2. Ensure both `index.html` and `assets/images/greg-getner-photo.png` are in the same directory structure
3. Open `index.html` in your browser

## Deployment

This is a static site that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service
